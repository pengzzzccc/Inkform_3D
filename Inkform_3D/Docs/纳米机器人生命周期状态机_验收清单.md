# 纳米机器人生命周期状态机（游荡/延伸/附身）+ 玩家角色化 验收清单

> 关联需求：补齐生命周期三状态 游荡 Wander ↔ 延伸 Extend ↔ 附身 Possess（切换都经过延伸）；

## 状态机

```
        Shift 扫描+E确认           Shift 扫描+E确认(换目标)
 游荡 ───────────────► 延伸(正) ───► 附身 ──┐
  ▲                                  │     │ 附身中 Shift+E
  │  延伸(反, 聚合成团)               │ E   └──► 延伸(正) ──► 附身
  └──────────────────────────────────┘ 脱离
```
- 切换**全部经过延伸**；延伸期间玩家被冻结（kinematic）。

## 怎么验收

### A. 自动化（EditMode 单测）
Test Runner ▸ EditMode ▸ Run All，`NanobotTreeTests` 全绿。新增：
| 测试 | 验收 |
|------|------|
| `TreeBranchFormation_GrowthFront_ForwardAndReverse` | 正向 t0=起点·t1=叶（延伸）；反向 t0=叶·t1=起点G（脱离聚合）→ 方向正确 |

（第一阶段的不回聚/均匀/确定性/贴面等断言继续保留。）

### B. 手动（`Assets/Scenes/NanobotSandbox.unity`，Play）
1. **游荡**：蜂群成团跟随；WASD 可移动；按 **Space 不跳**。
2. **游荡→附身**：Shift 扫描高亮（Shift 再按切候选）→ 按 **E** → 触手空中树状生长、表面包裹 → 进入附身。
   - [ ] 附身后物体**随 WASD 移动**、相机跟随、蜂群贴着物体表面一起动。
   - [ ] 按 **Space 可跳**。
3. **附身→附身**：附身中 Shift 扫描新目标 → 按 **E** → 从旧物体延伸到新物体；旧物体留原地、外观还原。
4. **附身→游荡（脱离）**：附身中（无高亮）按 **E** → 多条分支**向物体周围一个地面点聚合成团** → 回游荡、**不可跳**。
5. 选中 `NanobotSwarm` 看 Gizmos 预览延伸/聚合树（绿线=分叉、橙点=末梢）。

### C. 关键参数（`PossessionDirector` Inspector）
`DetachGroundRadius`(聚合点离物体半径) / `PlayerGroundOffset`(落地抬升) / `WrapTriggerProgress`(上色触发) / 第一阶段树参数沿用。

## 备注与已知取舍
- 驾驶时物理碰撞用玩家胶囊（附身期间关掉物体自身 Collider），物体按胶囊体积碰撞——沙盒可接受。
- 可附身物体无 Rigidbody：脱离后若原先被举在高处，会停在原地不下落（沙盒可接受）。
- 相机恒定跟随控制体 Player；状态切换的传送由 Cinemachine 阻尼平滑。
- 旧场景里 PossessionDirector 的已移除字段为残留键，重建场景或 Unity 重新序列化后清理；新字段取默认值。
